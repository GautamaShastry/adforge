# AdSynth AI

AdSynth AI is a full-stack, serverless, multimodal ad-generation platform that transforms a product image into a complete marketing asset using computer vision and generative AI.

## Tech Stack
- React (Frontend)
- AWS Lambda + API Gateway
- AWS Step Functions
- Amazon Rekognition
- Amazon Bedrock
- Amazon Polly
- Amazon S3
